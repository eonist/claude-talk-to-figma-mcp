About Conduit Conduit Plugin

This plugin allows an AI Agent to communicate with Figma, enabling AI-assisted design operations. Based on work by Sonny

Version: 0.5.3

How to Use

  1. Make sure the MCP server is running in the AI Agent
  2. Connect to the server using the port number (default: 3055)
  3. Once connected, you can interact with Figma through Claude
